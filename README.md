# adammanji.com

Personal site for Adam Manji — founder & engineer. Source for [adammanji.com](https://adammanji.com).

## Stack

- Next.js 16 (App Router, Turbopack)
- React 19, TypeScript
- Tailwind CSS v4
- Motion (`motion/react`) for animation
- Fraunces / Manrope / JetBrains Mono via `next/font/google`

## Local

```bash
npm install
npm run dev   # http://localhost:3000
```

```bash
npm run build
npm run start
```

## Deploy

Railway, auto-deploying from `main`.
